# Python Arknights-Style Analog Clock
PythonのTkinterライブラリを使用して作成した、アークナイツ（ロドス・アイランド）の世界観をイメージしたアナログ時計です。  
エンジニアへのキャリアチェンジに向けた、Python学習プロジェクトの一環として開発中。

## 🛠 特徴
- **数学的描画**: 三角関数（sin/cos）を用いた正確な針の動きと、文字盤の配置。
- **ロドス・デザイン**: 
  - 源石をイメージしたひし形フレーム。
  - 特徴的なセリフ体を用いた「RHODES ISLAND」ロゴ。
  - デジタルコンソール風の時刻・日付表示。
- **カスタマイズ性**: ウィンドウサイズやフォントの変更が容易な構造。

## 🚀 技術スタック
- Python 3.x
- Tkinter (標準ライブラリ)
